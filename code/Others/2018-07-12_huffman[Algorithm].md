# huffman编码

huffman编码是一种基于二叉树的常用压缩算法, 常用于压缩文本内容等. 

例如压缩文本"beep boop beer!" 

首先统计每个文字出现的频率

![img](../../img/2018071201.png)

然后从出现最少的文字开始构建节点, 将出现最少的两个连接同一个父节点, 然后再用来构建新树.
