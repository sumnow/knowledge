# 黑苹果开启hidpi

``` bash
    curl -o ~/enable-HiDPI.sh https://raw.githubusercontent.com/syscl/Enable-HiDPI-OSX/master/enable-HiDPI.sh
    
    chmod +x ~/enable-HiDPI.sh
    
    ~/enable-HiDPI.sh
```

重启, 使用RDM(见附件)选择HiDPI分辨率.
