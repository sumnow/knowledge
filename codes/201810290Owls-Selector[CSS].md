# Owls selector

猫头鹰选择器是如 `* + *` 这样的选择器

``` css
    *+* {
        margin - top: 5 px;
    }
```

+: 相邻兄弟选择器(Adjacent sibling selector)可选择紧接在另一元素后的元素, 且二者有相同父元素. 

这个就避免了首尾的多余margin, 好好利用吧, 骚年.
